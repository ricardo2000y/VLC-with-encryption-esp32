This file provides the external definitions for the inline GPIO functions defined in gpio_direct_RW.h. These definitions ensure that non-inlined versions of the functions are available if needed by the linker.

void directWriteHigh_single ( void )
externinline

External definition for directWriteHigh_single function.

Set a single GPIO pin to high state.

This external definition is provided to satisfy the linker in case the function is not inlined at all call sites. It sets the specified GPIO pin to high.

This function uses an assembly instruction to directly set a GPIO pin high, providing the fastest possible pin manipulation.

Note
This function is always inlined for maximum performance.

uint32_t gpioDirectRead ( void )
externinline

External definition for gpioDirectRead function.

Read the state of GPIO pins directly.

This external definition is provided to satisfy the linker in case the function is not inlined at all call sites. It reads the state of the specified GPIO pin.

Returns
uint32_t The state of the GPIO pin (0 or 1).

This function uses an assembly instruction to read the current state of all GPIO pins in a single operation.

Returns
uint32_t A 32-bit value representing the state of all GPIO pins.
Note
This function is always inlined for maximum performance.
